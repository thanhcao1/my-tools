I. System Preparation & Installation

‚ö†Ô∏è Important Note: You must disable antivirus software (Windows Defender) before extracting and running the tool to prevent system files from being deleted.
1. Install EasyOCR:
Copy the .EasyOCR folder to the following path on your computer: C:\Users\<Your_PC_Name>\.EasyOCR
2. Emulator Configuration (MuMu Player):
  • Screen Resolution: 1280 x 720
  • In-game Language: English
  • Enable ADB in emulator settings (See Section II below).

II. ADB Connection & Port

Step 1: Open CMD from Emulator Folder
Go to your emulator's installation folder. In the folder's address bar, type CMD and press Enter.
Step 2: Check Connection
In the CMD window, type the following command to see the list of connected ports: adb devices
Step 3: Enter Port into the Tool
In the Tool interface, find the Connection field -> Enter the Port number displayed in CMD -> Click Connect.
Check the Dashboard tab; if the emulator specs appear, the connection is successful.
